# Finance Dashboard

A React + Vite finance dashboard with summary metrics, charts, transaction management, import/export tools, and local storage persistence.

## Scripts

- `npm run dev` starts the Vite development server.
- `npm run build` creates a production build.

## Main Features

- Dashboard overview with summary cards and trend/category charts
- Transaction filtering, sorting, create/edit/delete flows, and CSV or JSON import/export
- Insights and settings pages with theme and local data controls
- Admin and viewer role switching stored locally in the browser

## Stack

- React 18
- TypeScript
- Vite
- React Router 7
- Tailwind CSS 4
- Recharts
- TanStack Table
- Radix UI primitives
